表头和内容不对齐
在使用antd的table组件时，由于需求需要用到表头固定的table组件，示例用法是设置其中某一列的width就可以自动固定住表头，同时由于内容行数很多，还需要可以上下滚动，所以还要设置 scroll= y : 600，实际用的时候由于列比较多，一旦使用 scroll 属性，就会发现表头和内容的列宽度根本没有对齐，而且由于table列宽默认是自动计算的，改变浏览器大小的话，列宽也都会自动改变，而且还是不对齐 解决办法123456789101112 table {     table-layout:fixed; } .resultColumns {     ...

允许长数字或者长单词自动折断
默认情况下，antd 的 table 组件里面的内容，遇到长单词或者长数字，都是为了完整性，默认无法折断的 解决办法在 table 外组件上添加属性： 1whiteSpace: 'normal', wordBreak: 'break-all', wordWrap: 'break-word'
